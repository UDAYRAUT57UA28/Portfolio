# Resume Folder

Place your resume PDF file here with the name `resume.pdf`.

## Instructions:

1. Save your resume as a PDF file
2. Rename it to `resume.pdf`
3. Place it in this folder (`assets/resume/`)
4. The download button on your portfolio will automatically work

## File Requirements:

- File name: `resume.pdf` (exactly as shown)
- Format: PDF
- Recommended size: Under 2MB for faster loading

Once you add your resume file, visitors can download it by clicking the "Download Resume" button in the hero section of your portfolio.
