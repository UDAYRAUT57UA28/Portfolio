# Profile & Avatar Image Instructions

## Adding Your Profile Photo

1. Take or choose a professional photo of yourself
2. Recommended specifications:
   - Size: 500x500px or larger
   - Format: JPG or PNG
   - Aspect ratio: Square (1:1)
   - File size: Under 500KB for faster loading

3. Name your photo `profile.jpg` or `profile.png`
4. Replace the existing `profile.jpg` file in this folder

## Adding Your Avatar/Illustration

1. Create or commission a tech-themed avatar illustration
2. Recommended specifications:
   - Size: 500x500px or larger
   - Format: JPG or PNG
   - Aspect ratio: Square (1:1)
   - Style: Tech/cyberpunk theme with coding elements
   - Background: Solid color or gradient

3. Name your avatar `avatar.jpg` or `avatar.png`
4. Replace the existing `avatar.jpg` file in this folder

## Avatar Creation Tools

- **Canva**: https://www.canva.com/ (Free templates)
- **Avatar Maker**: https://www.avatarmaker.com/
- **Picrew**: https://picrew.me/
- **Fiverr**: Commission a custom avatar from artists
- **AI Tools**: Use DALL-E, Midjourney, or Stable Diffusion

## Avatar Style Tips

- Tech/cyberpunk aesthetic
- Coding elements in background (code snippets, terminals)
- Futuristic accessories (VR headset, tech glasses)
- Purple/blue color scheme to match portfolio
- Professional but creative look

## How It Works

The profile photo will flip to reveal the avatar every 8 seconds:
- Front side: Your real photo
- Back side: Your tech avatar
- Smooth 3D flip animation
- Continuous loop

## Tips for a Great Profile Photo

- Use good lighting
- Plain or simple background works best
- Face should be clearly visible
- Professional attire (optional but recommended)
- Smile naturally
- Center yourself in the frame

The portfolio will automatically display both images with an animated 3D flipping effect!
